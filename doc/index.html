Embedded Web Server

EWS is a web server construction kit, designed for embedded applications using the GNAT Ada compiler.

Other Ada web-related software is linked here.

The project is hosted on SourceForge. You can see the releases, and you can browse the Mercurial repository.

Documentation will be forthcoming.

For the moment, and provided you are running on a Linux or Mac OS X system, or on Windows with Cygwin (other Unix systems should work as described, and the software runs on Windows but the makefile doesn't),

  • Install GNAT and XML/Ada;
  • Download the Ada 95 Booch Components, unpack, install according to the directions in the INSTALL file.

Then, in the EWS distribution's source directory (ews-yyyymmdd/src), 

   $ make demo
will create a server (ews-server-test), which when executed will listen on port 8080 and respond with these pages.

If you're seeing this page via the demonstration (ews-server-test), you can view a completely dynamic page here, and a page with AJAX content here.
