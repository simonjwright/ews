Embedded Web Server

EWS is a web server construction kit, designed for embedded applications using the GNAT Ada compiler.

The project is hosted on SourceForge. There are no releases yet, but you can browse the CVS repository.

Documentation will be forthcoming! For the moment, in the distribution directory (ews-yyyymmdd/), and provided you are running a Linux system and have GNAT installed, 

   $ make demo
will create a server (ews-server-test), which when executed will listen on port 8080 and respond with these pages.

The server should build and run fine under Windows, it's just the makefile that can't cope yet!

If you're seeing this page via the demonstration (ews-server-test), you can view a dynamic page here.
